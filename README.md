# PawMart — Pet Adoption & Supply Portal

---

Live Client: https://pet-care-8ba14.web.app/

Live Server: https://paw-mart-two.vercel.app/

---

PawMart is a community-driven platform built for pet owners, breeders, and pet shops to list pets for adoption and sell pet-related products. Users can browse listings, view details, adopt pets, place orders, and manage their own listings—all through a clean and modern UI.

---

## Features

Pet adoption and pet supply marketplace with category filtering.

Secure authentication using Firebase (Email/Password and Google login).

Add, manage, and order pets or products.

Download order reports as PDF using jsPDF & AutoTable.

Modern UI with animations using Tailwind CSS, Framer Motion, and Typewriter effect.

Dark/Light mode toggle (bonus feature).

Full CRUD operations with toast notifications.

Fully responsive across mobile, tablet, and desktop devices.
